#  Configurazione (Config)

Questa cartella contiene i file di configurazione essenziali per il funzionamento dell'applicazione.

- `db.php`: Contiene la connessione PDO al database MySQL.
- **Attenzione:** Se stai lavorando in locale, assicurati che i parametri (Host, Username, Password, Database Name) in `db.php` corrispondano a quelli del tuo ambiente XAMPP.
